Usage
=====

Start and play !
----------------

.. code:: bash

  $ npm start

You can now play with OpenParty and loaded gameplays in your **web browser**.

If you are running the server locally, you can test with `http://localhost:3040`. To stop the server, just kill it with `CTRL+C` keys.
